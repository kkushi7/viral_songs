# What Makes a Song Go Viral? A Data Analysis of Billboard Hits
This project analyzes Spotify audio features to identify which song characteristics are most strongly associated with reaching the Billboard Top 100. Through exploratory data analysis and statistical comparison, the goal is to uncover patterns in musical attributes that may contribute to commercial success.

Approach:
- Merge Spotify audio features with Billboard chart data
- Compare hit vs non-hit songs
- Identify statistically meaningful differences
- Translate findings into business insights
